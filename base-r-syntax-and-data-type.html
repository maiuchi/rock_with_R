5 Base R syntax and data type

5.1 Data type

  • Character vector (character string and/ or factor): defined by using " "
    • (e.g.) "Jack", "Rose"
    • Cannot perform mathematical calculations
  • Numeric vector (double, integer etc...): numbers are written as is
    • (e.g.) 100, 200
    • Can preform mathematical calculation
  • Logical vector: TRUE or FALSE

5.2 Create a variable and assign a value

  • R is case sensitive
    • X and x are two different variables for R
    • Variable names cannot have a space in between
      • public health will not work as a variable name. Instead, you can make it as public_health, publicHealth etc...
    • You cannot start variable names with numbers
      • 7data will not work. Instead, name it data7
    • Variable names are better to be descriptive and easy to understand
      • Otherwise, it will confuse you in the end

  • if you want to make a variable x and assign 1 as a value
    • <- is the symbol we use to assign values in R
    • # is to make comments on an R script
      • Make a habit to write descriptive comments, because you will often forget what this code was for
      • Try to keep you script neat and clean. This will help you have less bugs and make it easier to debug
x <- 1
x #check the variable

5.5 Use functions

  • What is function?
    • a type of set procedure(s) to perform a task
  • Calculate the sum of x, y, and z
    • R has lots of R functions that you can use, which makes it easier to perform data cleaning/ analysis
    • You can also write your own functions if you'd like in the future!
x <- 1
y <- 4
z <- x + y

#sum up x, y and z using the sum() function
sum(x, y, z)
#JFYI, what if you want to write your own fuction to calcurate the sum? 
# a, b, c can be anything
mySum <- function(a, b, c){
  a + b + c #telling R to add a, b, and c
}

#if you plug in numbers to your mySum function... you can calculate the sum! 
mySum(300, 750, 1000)

5.6 Make a data frame

  • Make a data frame "df1"
    • (explanation 1): "variable" is a column name, c("", "") is to add multiple vairiables. This says assign x to the "variable" column, row 1; assign y to the "variable" column, row 2; assign z to the "variable" column, row 3
    • (explanation 2): "value" is a column name. This says that assign 1 to the "value" column, row 1; assign 4 to the "value" column, row 2; assign 10 to the "value" column, row 3
  • Calculate the mean of x, y and z
df1 <- data.frame("variable" = c("x", "y", "z"), #(explanation 1)
                   "value" = c(1, 4, 10)) #(explanation 2)

str(df1)

#calculate mean
mean(df1$value) #"$" is select "value column" from df1

5.7 Create a simple plot

  • Create a data frame "firstHalf"
  • Create a simple bar plot using the "barplot()" function
firstHalf <- data.frame("month" = c("Jan", "Feb", "Mar", "Apr", "May", "June"), 
                   "value" = c(150, 200, 15, 70, 300, 60)) 

#create a bar plot. names.arg is to label each bar
barplot(firstHalf$value, names.arg = c("Jan", "Feb", "Mar", "Apr", "May", "June"))
