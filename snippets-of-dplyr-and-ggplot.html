6 Snippets of dplyr and ggplot

To be honest, these two packages are the game changers. I use them almost everyday along with a couple of other packages.

6.1 What is dplyr?

dplyr is an R package that allows you to perform data manipulation easily.

6.2 Why is dplyr so handy?

  • Once you are out of your class assignments, you will need to do data cleaning and manipulation (wrangling) by yourself to perform a data analysis.
    • Real world datasets are usually messy unlike the dataset you get from your professors for your assignments (yes, they usually clean the data for you)
    • We say that data analysists spend about 70% of their project time on data cleaning/ preparations
  • The dplyr package helps you perform data cleaning tasks
  • Data transformation with dplyr

6.3 Filter/ subset data using dplyr

Let's say that you'd want to select only the May data from the airquality dataset we used in Chapter 5. You can do that pretty easily by using dplyr

  • Load the dplyr package
    • For the very first time, you need to install the package: install.packages("dplyr") or install.packages("tidyverse"")
    • To load a package, use library() function
  • Load the airquality dataset
  • Make sure the data structure
    • glimpse() is a dplyr function to check data structures
  • Check the range of temperature in this dataset using the table() function
  • Filter the data frame by May
    • %>% is called piping or pipe operator. This means that grab from previous action/ item
      • In this case, it is telling R to grab the airquality dataset and then select the data that the "Month" column is equal to 5
    • Be aware that we use double equal signs (==) to tell R that we need the value that is exactly the same as 5 in the "Month" column
      • If you use a single equal sign (=), R will think that you are asking to assign the value 5 to the "Month" column, which will give you an error message.
      • Single equal sign (=) is the same as (<-) for R
#load dplyr 
library(dplyr)

#load the dataset 
airquality

#check the data structure 
glimpse(airquality)

#check te range of temperature in this dataset
table(airquality$Temp)

#filter May data from the dataframe 
airquality %>% filter(Month == 5)

#if you want to make a data frame called "may" which has only May data 
may <- airquality %>% filter(Month == 5)

6.4 Filter data with multiple conditions using dplyr

Now you know how to filter the data using dplyr. But what if you want to know how many days in May, 1973 were hotter than 74 degrees?

  • Load dplyr package
  • Load airquality dataset
  • Filter May and Temperature > 74
    • Notice that you use two pipes for this operation This code tells R, grab the airquality data and select May. And then from there, select the days that are hotter than 74
    • logical operators
    Operator Description
    < less than
    <= less than or equal to
    > greater than
    >= greater than or equal to
    == exactly equal to
    != not equal to 
#load dplyr 
library(dplyr)

#load the dataset 
airquality

#filter May and temp > 74 from the dataframe 
airquality %>% 
  filter(Month == 5) %>% 
  filter(Temp > 74)

6.5 What is ggplot?

ggplot is an R package that allows you to create nice-looking data visualization pieces

6.6 Why is ggplot so handy?

  • Data visualization is an important piece of your data analysis project, because you need to tell your audience a story of what you found in a understandable and easily digestible manner
    • R can be your one-stop-shop for your data analysis project
      • ggplot will let you create graphs without using other software
  • You don't need to know design theory to make nice visualizations
    • ggplot's preloaded theme/ color schemes will help to create a nice visualization
  • If you are a design person, ggplot gives you to a freedom to customize your plots/ graphs
    • You can change font size, font color (using rbg or hex number), line types and more!
  • One you design a base "template" of your visual piece, you can reuse it over and over again by copying and pasting the code!
  • Good news for mappers, you can make maps using ggplot as well
    • Making a COVID-19 Map in R

6.7 Create your first line graph using ggplot

  • Let's keep working on the airquality data
  • Filter the May data from the dataset
  • Create a line graph which shows the relation of temperature and day
    • (explanation 1): ggplot() tells R which data and variable should be used in what way.
      • ggplot(data-frame, aes(x = , y = ))
      • In today's case, we are telling R, use the may data frame as data, set Day as x-axis, Temp as y-axis
    • (explanation 2): geom_line() tells R what type of graphs to use
      • geom_line() is a line graph.
      • There are many types of graphs you can make using ggplot: R Graph Gallery
      • ggplot function references
#load dplyr and ggplot
library(dplyr)
library(ggplot2)

#load the dataset 
airquality

#filter May from the dataframe 
may <- airquality %>% 
  filter(Month == 5)

#create a line graph using ggplot 
ggplot(may, aes(Day, Temp)) +  #(explanation 1)
  geom_line() #(explanation 2)

#load dplyr and ggplot

library(ggplot2)

#load the dataset 
airquality

#filter May from the dataframe 
may <- airquality %>% 
  filter(Month == 5)

#create a plot
ggplot(may, aes(Day, Temp)) +  
  geom_line(color = "#000080") + #(explanation 1) 
  labs(title = "Tempurature Trends in NYC (May, 1973)") + #(explanation 2)

6.9 Clean up a graph even more!

  • Better! But, let's clean up a little bit more!
    • According you Edward Tufte, "data graphics" should have a data-ink ratio close to 1 (Tufte, 1982)
      • data ink is "non-erasable core of a graphic"
      • Data-ink ratio \(= \frac{\text{data-ink}}{\text{total ink used to print the graphic}}\)
    • This suggest that more readable graphics have less elements that do not directly relate to data
      • (e.g.) background grid, tick marks etc...

  • Make your graph cleaner and readable
    • When you want to change non-data element of the graph, you can modify it by adding the function theme()
      • (explanation 1): panel.grid means the grid of your graph. element_blank() tells R to remove it
      • (explanation 2): panel.border means the borders of your graph. element_blank() tells R to remove it
      • (explanation 3): axis.line means the line of x and y axises. element_line() tells R to change the look of the lines. In this case, the code tells R, change the color to black and line thickness to 0.3
#load dplyr and ggplot
library(dplyr)
library(ggplot2)

#load the dataset 
airquality

#filter May from the dataframe 
may <- airquality %>% 
  filter(Month == 5)

#create a plot
ggplot(may, aes(Day, Temp)) +  
  geom_line(color = "#000080") + 
  labs(title = "Tempurature Trends in NYC (May, 1973)")+ 
  theme_bw() + 
  theme(panel.grid = element_blank(), #(explanation 1)
        panel.border = element_blank(), #(explanation 2)
        axis.line = element_line(color = "black", size = 0.3)) #(explanation 3)

6.10 Final R code for this exercise

  • When you write and run R code in R Studio, you don't need to load your packages each time you run code
    • Difference between install and load packages
      • Install packages: You only need to do it once when you don't have a package installed in your R
      • Load packages: You need to load them when every time you re-open R or want to use a package that is not loaded yet (package is already installed)
      • Make sure you install and load all the packages you need in the beginning of your code
#install packages (only if you don't have them installed)
install.packages("ggplot2")
install.packages("dplyr")

#load packages
library(ggplot2)
library(dplyr)

#since the airquality dataset is preloaded to R, check the structure 
glimpse(airquality)

#filter the data by May and make a dataframe called may
may <- airquality %>% 
  filter(Month == 5)

#create a plot 
ggplot(may, aes(Day, Temp)) +  
  geom_line(color = "#000080") + 
  labs(title = "Tempurature Trends in NYC (May, 1973)", 
       caption = 
         "source: the New York State Department of Conservation (ozone data) 
          and the National Weather Service (meteorological data)")+ 
  scale_x_continuous(limits = c(1, 31), #setting the starting point to 1 and end to 31
                     expand = c(0,0),  #expand plot area, make it start right at 1
                     breaks = scales::pretty_breaks(n = 29)) + #increased tick marks
  theme_classic() + 
  theme(
    plot.caption = element_text(color = "grey60", size = 6), #change the size and color of caption
    plot.margin = unit(c(20, 20, 20, 10), "pt") #give wider margin on the right side 
  )
